# Spotify Recommendation Project (Ongoing)

## Overview
This project uses the **Spotify API** to collect music data and build a recommendation system. Currently in development, it demonstrates data collection, processing, and preliminary analysis of tracks and genres.

## Features
- Collect new releases and track metadata from Spotify  
- Analyze user listening preferences  
- Preliminary recommendation logic using clustering / similarity measures  

## Tech Stack
- Python  
- Pandas / NumPy  
- Spotipy (Spotify API client)  
- Jupyter Notebook  

## Quick Start
1. Clone the repo:
```bash
git clone https://github.com/YOUR_USERNAME/spotify-project.git
cd spotify-project
````

2.	Install dependencies:
```bash
pip install pandas numpy spotipy
````

3.	Set up environment variables in .env:
```bash
SPOTIPY_CLIENT_ID=your_client_id
SPOTIPY_CLIENT_SECRET=your_client_secret
````

4.	Run your scripts or open the notebook:
```bash
jupyter notebook notebook/spotify_notebook.ipynb
````
